  • Recipeze.net and Other Projects

    I started building Recipeze in January 2020. The website is designed to shorten user's grocery lists by recommending recipes that have ingredients in common with the recipe the user is currently viewing. Recipeze was deployed in June 2020, and stayed in production until around January 2022. Recipeze was hosted with Digital Ocean, the frontend is built with React, the backend with Django, Nginx is used as a reverse proxy, and Recipeze uses PostgreSQL as the database. The code for the frontend is publicly available on Github.

    While at WVU, I also created a number of other projects for classes. The most notable being a platform to create a Choose Own Adventure story in Javascript, an Image Filtering project in C, and an Animated Search Project in Java.
